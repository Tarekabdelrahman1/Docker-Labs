# ITI - Infrastructure & DevOps Labs


## 📌 Overview
This repository contains the practical labs, projects, and assignments completed during the **Information Technology Institute (ITI)** intensive training program. The focus of these labs is on modern infrastructure architecture, containerization, and deploying scalable microservices.

## 🛠️ Technologies & Tools
* **Containerization Engine:** Docker, Docker Compose
* **Web Servers & Reverse Proxy:** Nginx, Apache (httpd)
* **Backend Frameworks:** PHP, Python (Flask)
* **Databases:** MySQL
* **Version Control:** Git & GitHub
